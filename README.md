# Backend-mini-proyecto

Backend Spring Boot conectado a PostgreSQL.

## Conexion esperada

- Base de datos: `localhost:5437`
- Backend: `http://localhost:8084`

## Arranque

Antes de iniciar este repo, levanta la base de datos del repo `Database`.

Luego inicia el backend desde `mini-proyecto` con:

```powershell
docker compose up --build
```
